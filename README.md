# Deep Learning Course Materials

Course materials for the Deep Learning course at the Computer Science Department of Amirkabir University of Technology.

Term: Spring 2025  
Lecturers: Dr. Rahmati and Dr. Shamsi

## Contents

- `assignments`: homework notebooks, reports, and assignment files.
- `project`: final project report, presentation, notebooks, and data files.

## Naming

Files and directories use lowercase `snake_case`. Homework files follow the `dl_hw_XX` pattern.

## Notes

The notebooks contain the implementation work for the assignments and project. PDF files include reports, assignment descriptions, and presentation material.
